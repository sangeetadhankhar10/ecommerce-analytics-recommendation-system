# 📊 E-commerce Analytics & Recommendation System

An end-to-end Machine Learning project that combines **product recommendation** and **business analytics** in a single interactive Streamlit application.

This system helps businesses understand customer behavior, identify top products, and generate intelligent product recommendations using collaborative filtering.

---

## 🚀 Live Demo
🔗 https://ecommerce-analytics-recommendation-system-kkmpcizdysqsrhhpl674.streamlit.app/

---

## 🎯 Key Features

- 🎯 Product Recommendation System (Collaborative Filtering)
- 📊 Product Insights Dashboard (Top products & demand trends)
- 🧠 Customer behavior analysis
- 🎨 Multi-page interactive UI with modern design
- ⚡ Real-time recommendations
- 🌐 Deployed on Streamlit Cloud

---

## 🛠️ Tech Stack

- Scikit-learn
